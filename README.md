
# springboot-vegetarian

For EEIT45 First Group to develop project.

資展國際 EEIT45 (2022/03/02 ~ 2022/07/29)

跨域JAVA班 (後端工程師班) 第一組的作品 - 愛蔬網 

## Getting Started 

此專案使用[ 前後端完全分離 ]的方式開發

前端採用  Vue.js 前端框架作為主要開發

版本為Vue 3

前端環境為 Node.js 16.x 穩定版本 

後端環境使用 SpringBoot 框架
